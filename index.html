Révisions CEJM

Les 3 Fonctions de l'État (Musgrave)

1. Allocation L'État intervient pour corriger les défaillances du marché :
  • Biens Publics (éclairage, défense)
  • Externalités (pollution)
  • Monopoles naturels
  • Asymétrie d'information
2. Redistribution Réduire les inégalités (Justice sociale).

Mécanisme : Impôts/Cotisations -> État -> Prestations sociales -> Revenu Disponible.
3. Régulation Stabiliser l'économie à court terme :
  • Lutte contre chômage
  • Lutte contre inflation
  • Soutien croissance

Objectifs : Carré Magique de Kaldor

Politique Structurelle (Long Terme)

Agit sur les structures profondes (> 5 ans).

  • Politique industrielle (Secteurs stratégiques)
  • Politique d'innovation (Aides R&D)
  • Politique de concurrence
  • Transition énergétique

Politique Conjoncturelle (Court Terme)

A. Budgétaire (Gouv)
  • Relance : + Dépenses / - Impôts
  • Rigueur : - Dépenses / + Impôts
B. Monétaire (BCE) Gestion via Taux Directeurs.
Baisse taux -> Crédit moins cher -> Croissance.
Hausse taux -> Crédit cher -> Baisse Inflation.
Limites & Contraintes
  • PSC (Europe) : Déficit < 3% PIB, Dette < 60% PIB.
  • Soutenabilité : Risque effet boule de neige de la dette.
  • Mondialisation : Risque de fuite des capitaux.

1. Droit de la Concurrence

Autorité de la Concurrence (FR) / Commission Européenne (UE)

Pratiques Anticoncurrentielles (INTERDITES)
  • Entente illicite (Cartel) : Accord sur les prix/marchés.
  • Abus de position dominante : Évincer les concurrents.
Sanction : Lourdes amendes (% CA).
Concurrence Déloyale Sanctionnée par le tribunal (Dommages-Intérêts).
  • Dénigrement : Critiquer un concurrent.
  • Confusion : Imitation (logo/nom).
  • Désorganisation : Débauchage massif.
  • Parasitisme : Profiter de la notoriété.
